Button

Buttons are the main action points in forms. Use them consistently for submit, cancel, and navigation actions.

Canada.ca guidance Portal guidance

Canada.ca guidance

Use the standard Canada.ca button styling with clear hierarchy and consistent spacing between actions.

Primary buttons should stand out while secondary actions remain visible but less prominent.

Example markup

<button type="submit" class="btn-primary">Save and continue</button>
<button type="button" class="btn-secondary">Back</button>

Portal guidance

Inside portals, confirm the portal theme for button color, border, and hover treatment.

Keep the action structure the same: one main call to action and optional secondary or tertiary controls.

Example markup

<button type="submit" class="btn-primary portal-style">Save and continue</button>
<button type="button" class="btn-secondary portal-style">Cancel</button>
