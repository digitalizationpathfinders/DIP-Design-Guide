Pattern

Wizard flow

Use a wizard when a form has multiple steps and users need progress indicators and clear navigation.

Progress and navigation

Show the current step, use a clear title for each stage, and provide a consistent way to move forward and backward.

Accessibility notes

Include status text for screen readers and keep focus on the main heading after each step change.

<nav aria-label="Progress">
  <ol>
    <li aria-current="step">Personal information</li>
    <li>Review</li>
    <li>Submit</li>
  </ol>
</nav>

Portal vs Canada.ca

Inside portals, the progress UI may reflect portal navigation patterns. Outside portals, follow Canada.ca content structure and use visible step summaries.
