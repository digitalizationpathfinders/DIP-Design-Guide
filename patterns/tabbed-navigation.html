Pattern

Tabbed navigation

Use tabs to separate related sections in a single form without overwhelming users with too much content.

When to use tabs

Use tabs for logically grouped sections that are part of the same task, such as personal details, preferences, and review.

Implementation notes

Keep the number of tabs manageable and label them clearly. Avoid tabs if users must revisit many fields frequently.

Style considerations

Inside portals, tabs may follow a portal-specific UI library. On Canada.ca, follow the content tabs approach with a clean layout.
