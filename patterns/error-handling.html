Error handling

Provide clear, accessible error messages and guide users to fix issues without losing entered data.

Message clarity

Phrase errors as actions to take, not just problem descriptions. Place messages near the relevant input and set focus appropriately.

Portal vs Canada.ca

Consider whether portals have specific error summary components or focus patterns and adapt accordingly.
