Text & translations

Writing conventions

Use plain, direct language, keep instructions short, and write for the user’s context.

Be clear and concise

Use short sentences and avoid jargon. Prefer active voice and keep the user as the subject.

Label and help text guidance

Write labels that describe the expected answer, and use help text only when needed.

  • Label example: "Send notifications to" rather than "Notification email address"
  • Help text example: "We’ll only use this address for service updates."

Translation-friendly phrasing

Write complete ideas with consistent terminology to make translation easier.
